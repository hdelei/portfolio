Descrição do Projeto

Pequeno aplicativo desktop para registro e controle de clientes com serviço de gravação e visualização de imagens remotamente. Este é um projeto de uso particular no ramo de segurança eletrônica. Até a edição deste html, foi utilizado com muita frequência no meu trabalho, substituindo planilhas de Excel.

Detalhes do Projeto

  • Plataforma: Plataformas compatíveis com JVM
  • Linguagem: Java e HTML com CSS
  • Requerimento: Java Virtual Machine
  • Source: https://github.com/hdelei/CamControl
  • Detalhe: criado no NetBeans. Possui BD SQLite
